# Code of Conduct

## Our Pledge
We want this project to be a respectful, inclusive, and constructive space for everyone.

## Expected Behavior
- Be respectful and professional.
- Assume good intent.
- Give constructive feedback.
- Focus on improving the project.

## Unacceptable Behavior
- Harassment or abusive language.
- Personal attacks.
- Discrimination of any kind.
- Spam or intentionally disruptive behavior.

## Enforcement
Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.

## Reporting
Report conduct concerns privately through the project security/support contact once published.
